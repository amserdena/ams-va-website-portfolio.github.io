SunLoom RiceDry+

A solar-powered, IoT-integrated rice dryer with real-time sensors and AI-based moisture prediction. Designed to automate and optimize rice drying using both solar and AC power, with a digital interface for environmental monitoring and control.

Isolation Forest and PCA-Based Anomaly Detection

Developed an algorithm using Isolation Forest and Principal Component Analysis to detect and remove anomalies in rainfall and water level datasets, improving data reliability for hydrological systems.

SunLoom RiceDry+ Project Design – 3D Design Using Thinkercad

Designed in Tinkercad, the SunLoom RiceDry+ is a functional 3D prototype showcasing my skills in conceptual design and product modeling. The model features a cylindrical drying chamber with a mesh interior for grain distribution, a conical top for sunlight intake and husk containment, and a built-in exhaust with gears for airflow optimization. Wheels were added for mobility. This project highlights my ability to use digital tools for visual communication, technical support, and product presentation—key skills in a virtual assistant role.

Chicken Inasal Vlog – Edited with CapCut & Customized Canva Template

This video project features a Chicken Inasal cooking vlog, fully edited using CapCut for clips, transitions, text overlays, and background music. The thumbnail and promotional graphics were created in Canva, where I customized templates to match the vlog’s branding. Despite using only free tools, I handled the entire post-production process—from trimming and sequencing to visual design—showcasing my ability to produce polished, engaging content efficiently. This reflects my adaptability and creativity in delivering high-quality media using accessible platforms.
